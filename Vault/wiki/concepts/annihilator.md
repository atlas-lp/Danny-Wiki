# Annihilator

For a subset U of a vector space V, the annihilator U^0 is the set of linear functionals in the dual space V' that map every element of U to 0. For a subspace U of a vector space V, its annihilator U⁰ is the set of all linear functionals in the dual space V' that map every vector in U to zero; the text proves this set is a subspace of V'.
