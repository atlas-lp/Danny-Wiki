# Bilinear Form

A function β from V × V to a scalar field F that is a linear functional in each slot when the other slot is held fixed. The set of all bilinear forms on a vector space V, denoted V⁽²⁾, is itself a vector space. A function on a vector space V that takes two vectors as input and produces a scalar, and is linear in each of its two arguments. A function from V x V to a field F that is linear in each of its two slots, representing the m=2 case of a multilinear form.
