# Categorical Data Type

A special pandas data type that uses an integer-based representation or encoding for columns containing repeated instances of a smaller set of distinct values to achieve better performance and memory use. A pandas data type that can yield substantial performance gains and use significantly less memory by using an integer-based codes array instead of an array of strings.
