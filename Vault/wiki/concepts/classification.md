# Classification

A supervised learning problem where the goal is to predict one of several options, which are called classes. A type of supervised learning task where the goal is to predict a class label, which is a choice from a predefined list of possibilities. A primary supervised learning task focused on predicting categorical labels, for which algorithms like k-nearest neighbors, logistic regression, and support vector machines are used.
