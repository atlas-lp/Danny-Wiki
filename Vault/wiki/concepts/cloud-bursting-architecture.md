# Cloud Bursting Architecture

A form of dynamic scaling that scales or “bursts out” on-premise IT resources into a cloud environment whenever predefined capacity thresholds have been met. An architecture that enables an on-premise service to 'burst out' and use resources from a cloud environment when its on-premise processing thresholds are exceeded.
