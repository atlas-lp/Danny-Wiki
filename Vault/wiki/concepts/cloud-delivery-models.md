# Cloud Delivery Models

A fundamental cloud computing concept that is discussed in detail from both the cloud provider and cloud consumer perspectives. The primary models for cloud service delivery (SaaS, PaaS, IaaS) which are differentiated by the level of control granted to the cloud consumer and the responsibilities of the cloud provider.
