# Containerization

A cloud-enabling technology that is compared with virtualization and involves in-depth coverage of its specific environments and components. A form of virtualization technology that allows for the creation of virtual hosting environments called containers without the need to deploy a virtual server for each solution. A virtualization technology used to deploy and run applications and services in an isolated environment without the need to deploy a virtual server for each solution. A form of operating system virtualization used to deploy and run distributed applications by grouping an application and its dependencies into a single portable unit called a package.
