# Convolutional Neural Network (CNN)

A type of neural network suitable for high-dimensional inputs like images, where smaller regression models are trained on patches of the input to detect local patterns. A type of neural network where a layer consists of multiple convolution filters that slide across an input image or volume to compute features.
