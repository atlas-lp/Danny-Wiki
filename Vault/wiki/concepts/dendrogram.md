# Dendrogram

A visualization tool for hierarchical clustering that can handle multidimensional datasets by plotting the hierarchical cluster similarities. A tree-like plot used to visualize the hierarchy of partitions created by agglomerative clustering, showing how clusters are merged based on distance.
