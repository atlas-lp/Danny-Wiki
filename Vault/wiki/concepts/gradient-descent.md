# Gradient Descent

An approach to minimize a function by picking a random starting point and taking small, repeated steps in the opposite direction of the gradient. An optimization technique used to fit a model by iteratively moving in the direction of the steepest descent of the error function to find its minimum. An optimization algorithm used to fit models by maximizing a function like the log likelihood in logistic regression.
