# Jordan Form

A special matrix representation into which every linear operator on a complex vector space can be put, a proof of which is given in the text. A matrix representation of an operator on a complex vector space that is a block diagonal matrix, where each block has an eigenvalue on the diagonal, 1's on the line directly above the diagonal, and 0's elsewhere.
