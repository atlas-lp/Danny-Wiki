# L2 Regularization

A type of regularization used by Ridge regression that penalizes the L2 norm, or the Euclidean length, of the model's coefficients (w). A default type of regularization used in linear models, which is contrasted with L1 regularization. A technique used in linear models to constrain model coefficients and prevent overfitting by keeping their magnitudes small.
