# Linear Models

A widely used class of models that make a prediction using a linear function of the input features, which can be used for both regression and classification. A class of algorithms recommended as a first approach to try, which performs well on very large and high-dimensional datasets. A class of models used for regression and classification that make predictions based on a linear function of the input features.
