# Logical Network Perimeter

A virtualized IT environment, formed by a virtual firewall and an isolated virtual network, that supplies and controls the connectivity of a data center. A mechanism that creates logical boundaries to ensure virtual server clusters are enclosed in a secure, connected environment and that relocated servers comply with regulations. A mechanism that provides levels of isolation to ensure that cloud consumer data undergoing relocation remains inaccessible to unauthorized parties.
