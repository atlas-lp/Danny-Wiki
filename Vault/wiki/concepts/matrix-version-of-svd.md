# Matrix Version of SVD

A representation of an M-by-n matrix A of rank m as a product BDC*, where B is an M-by-m matrix with orthonormal columns, D is an m-by-m diagonal matrix with positive numbers, and C is an n-by-m matrix with orthonormal columns.
