# Metadata

Data about data, such as the format of a file, the time of uploading, or the account from which it was uploaded. A component of a data warehouse that documents information such as the calculation method for computed fields, ensuring every element is sufficiently well-defined. Data about data itself, which is captured by web crawlers and used by applications such as web search engines.
