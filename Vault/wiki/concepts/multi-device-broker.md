# Multi-Device Broker

A mechanism containing mapping logic used to facilitate runtime data transformation, making a cloud service accessible to a wider range of cloud service consumer programs and devices. A mechanism that commonly supports mobile-enabled SaaS implementations to facilitate access from various devices.
