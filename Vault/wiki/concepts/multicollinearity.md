# Multicollinearity

An error that can cause regression algorithms to fail due to the presence of exactly redundant predictor variables, such as when factoring categorical variables into binary indicators. A condition where predictor variables have perfect or near-perfect correlation, which can make a regression model unstable or impossible to compute.
