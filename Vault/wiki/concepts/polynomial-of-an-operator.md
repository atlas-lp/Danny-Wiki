# Polynomial of an Operator

A polynomial p(z) can be applied to an operator T to define a new operator p(T) by substituting T for the variable z and replacing the constant term a₀ with the operator a₀I. An operator p(T) formed by applying a polynomial p to a linear operator T, which commutes with T, i.e., (p(T))(Tu) = T(p(T)u).
