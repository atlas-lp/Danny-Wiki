# Positive Operator

A self-adjoint operator T for which `<Tv, v>` is non-negative for all vectors v, which is equivalent to having non-negative eigenvalues. Every positive operator has a unique positive square root. An operator T is an invertible positive operator if and only if the bilinear form defined by <u, v>_T = <Tu, v> is an inner product.
