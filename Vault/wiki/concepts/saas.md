# Software-as-a-Service (SaaS)

A cloud delivery model where the consumer is granted control over usage and usage-related configuration, typically by accessing a front-end user-interface. A cloud delivery model where cloud service architectures are generally based on multitenant environments that enable and regulate concurrent cloud consumer access. A cloud delivery model where a cloud consumer's control is generally limited to the runtime usage of cloud service instances, including managing user accounts and security configurations. A cloud delivery model where pricing is determined by application software usage, such as the number of application modules, nominated cloud service consumers, and the number of transactions.
