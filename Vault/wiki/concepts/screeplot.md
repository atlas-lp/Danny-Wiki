# Screeplot

A plot of the variances of the components, showing the relative importance of the components. A bar chart used with PCA that shows the amount of variance explained by each principal component, which helps in selecting the number of significant components to retain. A bar chart used to visualize the variance explained by each principal component in a Principal Component Analysis.
