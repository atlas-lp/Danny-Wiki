# Square Root of an Operator

An operator R is a square root of an operator T if R^2 = T. The text proves that every positive operator has a unique positive square root, denoted as sqrt(T). An operator R is a square root of an operator T if R^2 = T; invertible operators on a complex vector space have a kth root for every positive integer k.
