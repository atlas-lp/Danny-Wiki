# Stacked Bar Graph

A type of bar graph where values of multiple variables are stacked one on top of the other, which can be normalized to show relative composition. A chart that shows the percentage of components within categories for each item, such as the proportion of order sizes (Tiny, Small, Medium, Large) for each product.
