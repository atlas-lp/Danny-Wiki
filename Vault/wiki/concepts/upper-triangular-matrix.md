# Upper-Triangular Matrix

A square matrix where all entries below the main diagonal are zero. An operator has an upper-triangular matrix representation if and only if the spans of the initial basis vectors form a chain of invariant subspaces. A type of matrix that, in the context of QR and Cholesky factorizations, has only positive numbers on its diagonal.
