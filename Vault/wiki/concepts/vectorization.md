# Vectorization

A term used by NumPy users to describe the ability to express batch operations on data without writing any for loops. A style of computing that involves expressing element-wise operations directly on arrays, for which ufuncs are fast wrappers. The practice of using array expressions to perform operations on whole arrays at once, which is a key performance feature of NumPy.
