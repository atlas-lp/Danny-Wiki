# Automated Scaling Listener

A mechanism that monitors a cloud service to determine if predefined capacity thresholds are being exceeded and then initiates a scaling process based on a policy. A component that monitors workload, network traffic, or storage requests against pre-defined thresholds and signals other components to initiate scaling actions.
