# Boston Housing dataset

A complex dataset with 506 samples and 105 derived features used to compare the performance of linear regression, Ridge regression, and Lasso regression. A dataset used as a realistic application to demonstrate how polynomial features and interactions can significantly improve the performance of a Ridge regression model.
