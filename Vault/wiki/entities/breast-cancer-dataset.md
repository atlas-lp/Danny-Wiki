# Breast Cancer dataset

A dataset used to illustrate how different values of the regularization parameter C affect the learned coefficients of a LogisticRegression model. A dataset used to demonstrate feature importances from a gradient boosting model and to show the significant impact of feature scaling on SVM performance. A dataset containing 30 features used to classify cancer into two classes: benign and malignant.
