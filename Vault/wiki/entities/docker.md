# Docker

A common containerization technology discussed in the book. A containerization technology whose architecture can be divided into four key areas: Docker Server, Docker Client, Docker Registry, and Docker Objects. A containerization engine whose container functionality is enhanced by Kubernetes to provide an enterprise-grade architecture.
