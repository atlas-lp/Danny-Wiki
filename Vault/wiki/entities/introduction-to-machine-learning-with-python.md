# Introduction to Machine Learning with Python

A book for data scientists that teaches practical ways to build machine learning solutions using Python and the scikit-learn library. A book designed to help people apply machine learning in practice without requiring advanced math courses. The source book for the text excerpts, written by Andreas C. Müller and Sarah Guido.
