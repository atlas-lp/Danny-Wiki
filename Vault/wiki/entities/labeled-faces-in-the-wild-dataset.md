# Labeled Faces in the Wild (LFW) dataset

A dataset of grayscale face images of celebrities from the early 2000s, containing 3,023 images across 62 different people, used for face recognition tasks. A dataset of face images used as an example to apply and illustrate the components found by Non-negative Matrix Factorization (NMF). A real-world dataset of face images used as an example to apply clustering algorithms like DBSCAN to find structure and perform outlier detection.
