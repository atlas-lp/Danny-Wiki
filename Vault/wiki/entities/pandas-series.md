# pandas Series

A one-dimensional, array-like data structure in pandas that holds a sequence of values and an associated index of data labels, and can be thought of as a fixed-length, ordered dict. A one-dimensional pandas data structure whose indexing works analogously to NumPy array indexing but also allows the use of index values for selection.
