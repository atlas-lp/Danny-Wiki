# SciPy

A scientific Python library that the scikit-learn machine learning library depends on and is built on top of. A Python library that provides functionality for sparse matrices, such as the CSR and COO formats. A library that provides functions to compute a linkage array from data and to plot a dendrogram for visualizing hierarchical clustering. A Python library that provides the sparse matrix data structure used by scikit-learn to store bag-of-words representations.
