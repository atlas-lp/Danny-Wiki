# Apriori Algorithm

The most popular algorithm for association rule mining which finds frequent itemsets using the property that any subset of a frequent itemset must also be frequent. An algorithm used for association rule mining on nominal data that finds rules based on configurable minimum support and confidence metrics. A specific algorithm used to perform association rule mining.
