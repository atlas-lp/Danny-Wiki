# Business Agility

The measure of an organization's responsiveness to change, which can be improved by scaling IT resources through cloud computing. A benefit of cloud computing that empowers an organization to be more responsive by leveraging native cloud capabilities like on-demand scalability, automation, and reduced infrastructure maintenance.
