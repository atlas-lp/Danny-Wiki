# Cloud Delivery Model

A model concerning the different ways in which cloud services are administered and utilized by cloud consumers, such as IaaS, PaaS, and SaaS. A model that defines how cloud computing services are provided to consumers, with common types being IaaS, PaaS, and SaaS, which influence pricing structures.
