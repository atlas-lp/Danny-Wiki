# Cloud Usage Monitor

A lightweight and autonomous software program responsible for collecting and processing IT resource usage data. A monitor, often implemented as an extension to a VIM using its API, that collects runtime resource usage events and forwards them to other systems. A specialized monitor used to track LUN movements and collect workload distribution statistics. A mechanism used in IaaS and PaaS environments to track resource usage such as server lifecycles, storage, and network traffic for billing and SLA purposes.
