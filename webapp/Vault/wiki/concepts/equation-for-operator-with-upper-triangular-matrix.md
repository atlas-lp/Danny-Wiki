# Equation for Operator with Upper-Triangular Matrix (5.40)

A theorem stating that if an operator T has an upper-triangular matrix with diagonal entries λ₁, ..., λₙ, then the operator product (T - λ₁I)⋯(T - λₙI) is equal to the zero operator.
