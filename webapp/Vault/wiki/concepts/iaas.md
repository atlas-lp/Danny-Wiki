# Infrastructure as a Service (IaaS)

A cloud delivery platform for provisioning services such as virtual servers, which was chosen by the company DTGOV for its cloud initiative. A cloud delivery model that grants the consumer full administrative access to virtualized infrastructure-related IT resources and, possibly, to the underlying physical IT resources. A cloud delivery model where fundamental IT resources, such as virtual servers and cloud storage devices, are offered to cloud consumers. A cloud delivery model where consumers access virtual servers at the operating system level and have a high degree of control over provisioning IT resources like scalability, virtual networks, and storage. A cloud delivery model where pricing is usually based on IT resource allocation and usage, such as transferred network data, number of virtual servers, and allocated storage capacity.
