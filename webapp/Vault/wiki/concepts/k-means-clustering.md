# K-Means Clustering

A clustering algorithm that partitions inputs into a specified number (k) of clusters by iteratively assigning inputs to the nearest cluster mean and then recomputing the means. A clustering method that partitions data points into a specified number of clusters, used in the text to partition pixels in red-green-blue space for image recoloring.
