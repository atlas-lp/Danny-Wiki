# L1 Regularization

A type of regularization used by the Lasso model that penalizes the L1 norm of the coefficient vector, which is the sum of the absolute values of the coefficients. A type of regularization that can make a model more interpretable by limiting it to using only a few features. A technique used in linear models to constrain model coefficients, which can be useful for feature selection by shrinking some coefficients to zero.
