# Load Balancer

A runtime agent that distributes workloads across two or more IT resources to increase performance and capacity, and can perform specialized functions like asymmetric or content-aware distribution. A mechanism that can be added to horizontally balance workloads across available cloud storage devices or distribute requests across cloud service implementations.
