# Multi-Cloud Architecture

An architecture where an organization utilizes resources from different cloud providers to avoid vendor lock-in and select best-of-breed services based on geographical, economical, or operational advantages. An architecture where an organization identifies and selects services from different cloud providers to combine benefits and achieve an optimal economical advantage.
