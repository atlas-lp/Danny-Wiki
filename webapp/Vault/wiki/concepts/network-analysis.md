# Network Analysis

A data science technique demonstrated by analyzing a social network's users and their friendship connections, represented as a list of pairs of user IDs. A field of data analysis that models problems in terms of networks, which consist of nodes of some type and the edges that join them. A set of techniques for analyzing networks, including Betweenness Centrality, Eigenvector Centrality, and PageRank for directed graphs.
