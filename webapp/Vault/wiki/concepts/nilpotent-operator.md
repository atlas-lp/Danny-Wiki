# Nilpotent Operator

A linear operator that equals the zero operator when raised to some power. An operator `T` on a vector space `V` is called nilpotent if some power of it equals the zero operator, which implies that `T` raised to the power of the dimension of `V` is also the zero operator. An operator T for which T^m = 0 for some positive integer m. Every nilpotent operator on a real or complex vector space has a Jordan basis.
