# Norm

A value for a vector v, denoted ||v||, defined as the square root of the inner product of the vector with itself, which represents the vector's length or magnitude. A function from a vector space to non-negative real numbers that assigns a length to each vector, satisfying properties of being zero only for the zero vector, scaling with scalar multiplication, and the triangle inequality.
