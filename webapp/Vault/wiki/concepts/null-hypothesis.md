# Null Hypothesis

The default assumption in a statistical test that there is no effect or no difference, which the test aims to provide evidence against. A hypothesis that two probabilities are the same (e.g., that P_A - P_B is zero), which can be tested using a statistic. A component of statistical hypothesis testing, which is tested in procedures like an A/B test.
