# Platform as a Service (PaaS)

A cloud platform that can be used to migrate applications running on different technologies, as prototyped by the company ATN. A cloud delivery model that provides a ready-made environment and grants the consumer a moderate level of administrative control over IT resources relevant to their usage of the platform. A cloud delivery model that provides cloud consumers with ready-made environments, which include simulated runtimes and application servers, for deploying applications. A cloud delivery model that provides ready-made environments with selected software platforms and development frameworks, offering less administrative control than IaaS. A cloud delivery model with pricing similar to IaaS for network, server, and storage resources, but with variable prices depending on software configurations, development tools, and licensing fees.
