# Pivot Table

A data summarization tool that aggregates a table of data by one or more keys, arranging the data in a rectangle with group keys along the rows and columns. A data summarization method used to aggregate and reorganize data, demonstrated by calculating mean movie ratings for each film grouped by gender. A data summarization tool used to aggregate data by one or more keys, such as creating a table of campaign contributions with occupations as the index and political parties as the columns. A data summarization tool that aggregates and reshapes data, often used in a split-apply-combine pattern.
