# Polynomial

A function p from a field F to itself such that p(z) = a_0 + a_1z + ... + a_mz^m for some coefficients a_i in F. A function p from a field F to itself, defined by a sum p(z) = a_0 + a_1z + ... + a_m z^m for coefficients in F, where the highest coefficient a_m is non-zero.
