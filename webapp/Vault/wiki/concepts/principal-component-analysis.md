# Principal Component Analysis

A technique used to extract one or more dimensions that capture as much of the variation in the data as possible, and is particularly useful for dimensionality reduction in high-dimensional datasets. A method for dimensionality reduction, used in one project to find the principal components of shirt images, called 'eigenshirts', for classification.
