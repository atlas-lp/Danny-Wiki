# Resource Management System

A system that manages the lifecycle of cloud IT resources, including managing templates, allocating and releasing virtual resources, coordinating other mechanisms, and enforcing policies. A mechanism used to define usage thresholds for individual IT resources and resource pools, and to manage resource allocations. A system within a cloud provider's infrastructure that controls and manages cloud resources, often including a VIM and a VM image repository.
