# %timeit

An IPython magic command that runs a statement multiple times to compute an ensemble average execution time, which is useful for timing code with very short execution times. An IPython magic function that runs a statement multiple times to produce a more accurate and precise average runtime, useful for analyzing code with very short execution times.
