# VIM (Virtual Infrastructure Manager)

A platform that manages virtualized IT resources and can generate events about their status and usage, which are captured by monitoring agents. A management component that a failover system interacts with to redirect network traffic in response to server unavailability notifications. A component within a resource management system responsible for managing a cloud's physical servers and virtualized resources.
