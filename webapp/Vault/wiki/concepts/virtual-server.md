# Virtual Server

A central and primary IT resource within a typical IaaS environment that is leased by specifying hardware requirements like processor capacity, memory, and storage. A foundational building block of cloud environments, synonymous with Virtual Machine (VM), that can be instantiated from image files rapidly and on-demand.
