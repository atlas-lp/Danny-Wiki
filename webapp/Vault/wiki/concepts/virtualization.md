# Virtualization

A contemporary, cloud-enabling technology that helps realize modern-day cloud computing platforms and is often compared to containerization. The process of converting a physical IT resource into a virtual IT resource, severing the dependency between software and static hardware environments. A technology used in data centers where IT resources can be virtualized, enabling IT resource consolidation and strategies like storage virtualization. A technology that enables multiple virtual servers to be created and share one physical server, supporting cloud features like on-demand usage, resource pooling, and elasticity. The technology that enables a physical IT resource, such as a physical server, to provide multiple virtual images of itself so its processing capabilities can be shared.
