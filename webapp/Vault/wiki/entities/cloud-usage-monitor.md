# Cloud Usage Monitor

A mechanism that carries out runtime workload tracking, data processing, and monitors IT resource consumption levels. A monitoring mechanism that tracks and collects usage information for various cloud resources, such as direct LUN access, elastic network capacity, and data transfers.
