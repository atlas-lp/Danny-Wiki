# F^n

A vector space over a field F consisting of n-tuples (x_1, ..., x_n), which served as the motivation for the definition of a vector space. A vector space shown to be finite-dimensional as it is spanned by the list of standard basis vectors (e.g., (1,0,...), (0,1,...), etc.).
