# Fundamental Theorem of Linear Maps

A theorem for a linear map T from a finite-dimensional vector space V, which provides the equality dim(range T) = dim(V) - dim(null T). A theorem stating that for a linear map T on a finite-dimensional vector space V, the dimension of V equals the sum of the dimensions of the null space and the range of T.
