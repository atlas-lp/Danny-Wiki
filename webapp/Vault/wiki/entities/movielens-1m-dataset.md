# MovieLens 1M dataset

A dataset used as an example to demonstrate creating indicator variables for data where a row can belong to multiple categories, such as a movie having multiple genres. A dataset containing 1 million ratings from 6,000 users on 4,000 movies, organized into three tables for ratings, users, and movies. A specific dataset containing movie ratings that is used as a practical example for data analysis techniques in the book.
