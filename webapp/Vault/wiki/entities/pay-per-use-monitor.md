# Pay-Per-Use Monitor

A mechanism that collects usage and billing information on how cloud consumers are allocated and use IT resources from pools, especially as it fluctuates with elastic provisioning. A monitoring mechanism that collects and classifies usage cost information for cloud resources, such as direct LUN access or storage on different performance tiers.
